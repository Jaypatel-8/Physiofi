# Condition icons (PNG)

Place one PNG per condition, named by **slug**. The conditions page loads them from here.

| Filename | Condition |
|----------|-----------|
| `low-back-pain.png` | Low Back Pain / Sciatica |
| `neck-pain.png` | Neck Pain / Cervical Spondylosis |
| `shoulder-pain.png` | Shoulder Pain |
| `knee-pain.png` | Knee Pain |
| `sports-injuries.png` | Sports Injuries |
| `post-operative.png` | Post-Operative Rehabilitation |
| `stroke-rehabilitation.png` | Stroke Rehabilitation |
| `parkinsons.png` | Parkinson's Disease |
| `spinal-cord-injury.png` | Spinal Cord Injury |
| `copd-asthma.png` | COPD / Asthma / Breathing Issues |
| `post-covid.png` | Post-COVID Recovery |
| `pediatric-developmental.png` | Pediatric Physiotherapy |
| `torticollis.png` | Torticollis (Children) |
| `balance-problems.png` | Balance Problems (Geriatric) |
| `osteoporosis.png` | Osteoporosis |
| `pregnancy-pain.png` | Pregnancy-Related Pain |
| `urinary-incontinence.png` | Urinary Incontinence |

Recommended size: **96×96 px** or **128×128 px** (they are shown at 48×48 in the UI). If a file is missing, a fallback checkmark icon is shown.
